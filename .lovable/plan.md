
## План: финальный порядок и текст email-блока

### Структура экрана (сверху вниз)

```text
┌─ Книга + описание + дедлайн/цена
├─ Темы главы (1-2 выбор)
│
├─ ☐ Я согласен на обработку ПД
│
├─ [ОРАНЖЕВАЯ CTA] Обсудить тему с редактором
├─ [ссылка] Нет Telegram? Написать в MAX
│
└─ ┌─ 📧 Хочу получить концепцию книги на почту
   │     и подробнее ознакомиться с условиями
   │   [email input] [Отправить концепцию]
   │   ☐ Хочу получать полезные материалы
   └─
```

### Изменения в `src/components/quiz/QuizResults.tsx`

1. **Порядок JSX:** Topics → PD consent → CTA block → Email block.
2. **CTA-блок всегда виден** — снимаем условие `{emailSubmitted && ...}`. Гейт по `consentPD` сохраняется.
3. **Email-блок:**
   - Заголовок: «Хочу получить концепцию книги на почту и подробнее ознакомиться с условиями»
   - **Подзаголовок «Персональный разбор: тема, структура…» — удалить полностью**
   - Контейнер мягче: `bg-muted/40 border-border/50` + иконка `Mail` (lucide) слева от заголовка
   - Кнопка «Отправить концепцию» гейтится `consentPD`
4. **Логика без изменений:** webhook-события `email_submitted` и `cta_bot_click` остаются.
